Realizzare i seguenti programmi in linguaggio C

Realizzare nel proprio codespace una cartella chiamata PausaDidattica dove inserire i file sviluppati durante queste lezioni

**e01.c**
Realizza un programma C che richiede in input tre valori interi maggiori di zero,
determina e comunica il valore minimo e massimo.

**e02.c**
Realizza un programma C che richiede un valore N intero da tastiera, se compreso tra 1 e 15,
genera N numeri casuali e per i numeri pari mostra anche i divisori.

**e03.c**
Realizzare un programma C che richiede un numero N da tastiera, se compreso tra 1 e 35, 
calcola quanti passaggi occorrono per superare il valore 100, seguendo questo schema:
Se N è un numero pari allora somma a se stesso la sua metà, se N è un numero dispari aggiunge 1.
