Le esercitazioni prevedono che sia installato e funzionante un server web come Xampp, Wampp, Easyphp.
Le cartelle presenti nel repository devono essere poste direttamente nella cartella root del server installato "\htdocs" oppure "\www"
o all'interno di una sottocartella della root "\htdocs\Esercitazioni\".
